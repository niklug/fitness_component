Multi Calendar Component

This is the main window for the multi calendar component, where you can perform the main operations to manage calendars. In the top bar, below the Joomla! main menu, you will find the buttons for the main operations:

Publish:

Set as Public the calendars selected in the main list.

Unpublish

Set as non-public the calendars selected in the main list.

Delete

Erase the calendars selected in the main list.

Edit

Edit the title of the calendar selected in the main list, details in the Help option under Edit section.

New

Create a new calendar, see details in the Help option under New section.

Help

Access the Help directory.
