Manage Calendar Content

In this window you can perform the main operations on the contents of the selected calendar. On the toolbar buttons are the main operations:

Add Multi

Open a popup in the admin page where you can enter details of the new multi to add. It is mandatory to choose a date range and color, use the calendar to select the date range in which the multi will take place and select the color with which the cells will be marked for the range of dates for this multi on the calendar.

Multi List

Displays a popup in the admin page where you can see the full list of multis related to this calendar. You can choose to display all multis or only the current month's multis.

Edit an multi

To edit the content related to an multi it is necessary to click on any of the regions with the color that corresponds to the multi. This will bring the same pop-up used for their creation where you can redefine all the details.

Delete an Multi

The operation to delete a calendar may be held from the editor window or from the list. Be careful, once confirmed the transaction may not recover erased data.
